## About Me 
I’m a creative technologist and Columbia University student working at the intersection of interactive media, systems design, and storytelling. I build installations, prototypes, and hybrid hardware–software works that translate complex social, historical, and speculative ideas into tangible experiences.

My work explores how systems - technical, cultural, and material - shape how people perceive, remember, and interact with the world.

I am especially interested in installation art, speculative design, and interactive systems as expressive mediums.

***Selected works focus on interactive systems, material computation, and narrative-driven design.***


## Interactive Systems - software-based works
 - [Retro Media Emporium](https://github.com/melanielaporte/Retro-Media-Emporium): Interactive System, a digital library for tracking, exploring, and sharing physical media among friends, that combines data modeling, a responsive frontend, and intuitive UX.
 - [SysAdmin Automation](https://github.com/melanielaporte/fieldkit-automation) a Linux/Raspberry Pi setup that provisions users, deploys software, and collects logs in a real-world system automation.
 - [SustainTulsa](https://github.com/melanielaporte/SustainTulsa) a HTML/CSS/JS coded frontend project, responsive UI design
 - [Pirate Memory Game](https://github.com/melanielaporte/Pirate-Memory-Game) a tiny interactive game, coding, UX, and playful interface design.
